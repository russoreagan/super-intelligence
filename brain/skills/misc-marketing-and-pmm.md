# Marketing & PMM (Definitive)

## Goal
Turn product value into **clear messaging** and a **go-to-market plan** that drives pipeline and adoption.

## Core outputs
- ICP and segments\n+- Positioning statement (April Dunford-style)\n+- Messaging hierarchy (value prop → benefits → proof)\n+- GTM plan (channels, budget, timeline)\n+- Launch plan (internal + external)\n+- Competitive intel + battlecards\n+
## Workflow
### 1) Define ICP
- Firmographics + technographics + psychographics.\n+- Identify buyer personas (economic, technical, champion).\n+- Validate against real customer data.\n+
### 2) Positioning (April Dunford)
- Competitive alternatives (including status quo).\n+- Unique attributes → value → best-fit customers.\n+- Choose market category (head-to-head vs niche).\n+- Add “why now” trends.\n+
### 3) Messaging architecture
- One-liner value proposition.\n+- 3–5 benefits with outcomes.\n+- Features as evidence.\n+- Proof points (case studies, metrics, logos).\n+- Adapt per persona.\n+
### 4) GTM + demand gen (multi-channel)
- Map TOFU/MOFU/BOFU tactics.\n+- Plan channel mix (digital + TV/CTV).\n+- Define KPIs (CAC, pipeline, ROAS, lift).\n+- Ensure attribution setup (UTMs, CRM/HubSpot, TV attribution).\n+
### 5) Competitive intelligence
- Create battlecards: positioning, differentiation, objections, traps.\n+- Feed market-research outputs into messaging updates.\n+
